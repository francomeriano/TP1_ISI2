## 🐍 Programas Implementados

### 1. Números Primos (`src/primes.py`)
- Verifica si un número ingresado es primo o no
- Incluye comentarios explicativos en el código

### 2. Cálculo de Factorial (`src/factorial/factorial.py`)
- Calcula el factorial de un número
- Soporta rangos como `4-8`, `-10` y `5-`
- Versión orientada a objetos: `factorial_OOP.py`

### 3. Conjetura de Collatz (`src/collatz/collatz_plot.py`)
- Calcula la cantidad de iteraciones para números del 1 al 10000
- Genera un gráfico con matplotlib
